# 🧠 本地知识库 RAG

> 把你的文档变成 AI 可搜索的个人知识库
> ClawHub 缺什么：他们没有本地文件系统权限，无法读取你的本地文档构建知识库

## 核心能力
- **文档导入**：读取 PDF/DOCX/TXT/MD 等格式
- **知识提取**：自动提取关键信息、摘要、标签
- **语义搜索**：用自然语言搜索你的知识库
- **智能问答**：基于知识库回答你
- **自动分类**：按主题/项目/时间自动归类

## 工作流程
```
你的文档文件夹
   ↓
读取 + 分块 + 提取关键信息
   ↓
构建可搜索的知识索引
   ↓
随时提问 → 基于你的文档回答
```

## 使用方式
```
"帮我从这个PDF里提取关键知识点"
"搜索一下我文档库里关于[某某]的内容"
"基于我的笔记回答：...？"
"把这一批文档整理成知识库"
```
