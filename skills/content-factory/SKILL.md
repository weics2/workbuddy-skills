# 🏭 全自动内容工厂

> 输入一个主题，输出全套内容资产
> 独有能力：结合了搜索+写作+图片生成+语音合成+文档排版，其他 AI Agent 做不到全链路自动化

## 流水线

```
输入主题
   ↓
S1 深度调研 ──── 多引擎搜索 + 资料整理
   ↓
S2 内容创作 ──── 结构化写作 + 大纲 + 正文
   ↓
S3 视觉生成 ──── 多模态生成图片/图表/封面
   ↓
S4 语音输出 ──── edge-tts 生成语音版
   ↓
S5 排版交付 ──── 同时输出 DOCX + PPTX + PDF

交付：1 个主题 → 5 种格式文件
```

## 使用方式

```
"帮我出一份关于[主题]的全套内容"
```

自动执行：
1. 搜索调研 → 整理资料
2. 撰写完整文章/报告
3. 生成配图/封面（多模态内容生成）
4. 语音合成（edge-tts）
5. 导出 Word + PPT + PDF

## 适用场景
- 行业调研报告：调研→写作→PPT演示→语音版
- 自媒体内容：文章→封面图→音频→多平台适配
- 学习笔记：主题研究→笔记→思维导图→语音重温
