# 💻 桌面智能管家

> 自动整理你的电脑，还你一个干净的桌面
> 独有能力：能直接操作本地文件系统，ClawHub 等云端 Agent 做不到

## 核心能力

### 1. 桌面清理
- 将桌面文件按类型分类（文档/图片/视频/压缩包/安装包）
- 按日期归档（2026-01月/02月/...）
- 按项目归类

### 2. 下载文件夹整理
- 识别并归档已下载的文件
- 清理重复文件
- 删除临时文件（.tmp .log .cache）

### 3. 文件索引报告
- 生成完整的文件目录索引
- 标注大文件（>100MB）
- 标注长时间未使用文件（>90天）
- 标注可能的重复文件

### 4. 智能重命名
- 批量重命名为统一格式
- 提取关键信息作为文件名

## 使用方式
```
"帮我整理一下桌面"
"清理下载文件夹"
"生成文件索引报告"
"帮我找找有没有重复文件"
```
