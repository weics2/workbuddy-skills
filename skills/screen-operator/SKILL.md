# 🖱️ 桌面屏幕操作专家

> 模拟人手操作电脑——云端 Agent 这辈子都做不到
> ClawHub 缺什么：他们是纯云端程序，没有桌面、没有显示器、没有鼠标键盘

## 核心能力
- **鼠标控制**：移动、点击、拖拽、右键
- **键盘输入**：打字、快捷键、组合键
- **屏幕截图**：全屏/区域截图并分析
- **应用控制**：打开/关闭程序，操作界面
- **自动化操作**：录制回放、定时执行

## 前置安装
```bash
pip install pyautogui pillow
```

## 使用场景
- "帮我打开计算器，算一下..."
- "截取屏幕这部分区域的图"
- "自动填写这个表单"
- "每隔1小时帮我检查一下这个软件的状态"

## 注意
- ⚠️ 操作桌面需要你的明确授权
- 每次执行前会先问你确认
