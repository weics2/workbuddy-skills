# 📡 智能监控与自动报告

> 帮你盯着世界，到点自动汇报
> 独有能力：结合定时任务+搜索+写作+排版，全自动信息监控与报告生成

## 核心能力

### 1. 多源监控
- **技术动态**：监控 GitHub Trending、技术博客、RSS
- **行业新闻**：监控行业热点、竞品动态
- **个人关注**：天气变化、股票行情、特定事件
- **代码仓库**：Issue/PR 更新、CI 状态

### 2. 自动报告
- **日报**：每日早报，汇总昨日重要动态
- **周报**：每周总结，分析趋势变化
- **月报**：每月深度分析报告

### 3. 智能摘要
- 自动提取关键信息
- 去重和优先级排序
- 生成结构化摘要

### 4. 多格式交付
- Markdown 摘要
- 完整 Word 报告
- 语音简报（edge-tts）

## 使用方式
```
"每天早上8点给我一份AI行业早报"
"监控这个GitHub仓库的新Issue"
"每周日生成一份技术动态周报"
"帮我盯着这个博客的更新"
```
